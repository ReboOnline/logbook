How to Install Windows 11 on an Unsupported PC (with Rufus) – Full Guide

Are you running Windows 10 on an older PC and considering upgrading to Windows 11 despite the infamous compatibility checks?
Good news: it's absolutely possible, and this guide will walk you through it step-by-step – including how to bypass TPM and Secure Boot requirements using Rufus.
We'll start with some essential checks to ensure a smooth transition.



Step 0: Check Your System Compatibility

  1. Before diving into the installation, make sure your system meets the practical (not official) requirements for running Windows 11.
    • Press Win + R, type msinfo32 , and hit Enter.
  1. Look for BIOS Mode:
    • If it says UEFI – perfect!
    • If it says Legacy, you'll need to convert to UEFI before proceeding.

Retail License Check

  1. Open a terminal (cmd.exe) or powershell and type or copy:
    • slmgr /dli and click OK
    • If it says Retail, you're good to go. This means your license is transferable and will re-activate after reinstalling.


Step 1: Download the Windows 11 ISO

    • Go to the official Microsoft Windows 11 download page.
    • Scroll down to Download Windows 11 Disk Image (ISO).
    • Select the latest edition, click Download.
    • Choose your language and download the 64-bit ISO.


Step 2: Download and Start Rufus

    • Go to Rufus official site.
    • Download the latest portable version.
    • Launch Rufus (no install needed).


Step 3: Prepare Your USB Drive

    • Insert a USB stick with at least 8GB of space.
    • All data will be erased, so back up if necessary.


Step 4: Configure Rufus

    • Device: Select your USB stick.
    • Boot selection: Click Select and choose the Windows 11 ISO.
    • Image option: Leave it at Standard Windows Installation.
    • Partition scheme: Choose GPT.
    • Target system: Ensure it's UEFI (non-CSM).
    • Click Start.

      • A pop-up will appear with custom options:

    • ☑ Remove requirement for 4GB+ RAM, Secure Boot, and TPM 2.0
    • ☑ Remove requirement for online Microsoft account (optional – leave unchecked if you plan to use your MS account)

      • Click OK to start the USB creation process.



Step 5: Boot from USB & Install Windows 11

    • Restart your PC.
    • Enter the Boot Menu (commonly via F12, Esc, or Del during startup).
    • Choose the USB stick as the boot device.
    • Follow the installation prompts:
      • Choose Custom install (not upgrade)
      • Delete all partitions on your drive (if doing a clean install)
      • Let the installer do the rest!


Step 6: Activate Windows 11

  • If you:
    • Use the same Microsoft account as before,
    • And previously had an activated copy of Windows 10,

      • Then Windows 11 will activate automatically once you connect to the internet.
      You can check activation under: Settings ➔ System ➔ Activation



🚫 No TPM or Secure Boot? No Problem!

  • Thanks to Rufus, these requirements are bypassed during install. You don't need to modify your BIOS or do anything risky. However:
    • Your system may receive a warning under Settings
    • Microsoft may block future updates on unsupported hardware – but as of now, security updates continue to arrive


😊 Conclusion

  • You now have a fully working Windows 11 installation on hardware that Microsoft technically doesn’t support
    • but performs just fine.
  • Tips:
    • Keep your Windows 10 backup handy (use tools like Macrium Reflect if you want a full image)
    • Make sure you use an SSD for smooth performance
    • Enjoy the cleaner UI, Snap Layouts, and other modern features in Windows 11!
