How to Install Windows 11 on an Unsupported PC (with Rufus) – Full Guide

Are you running Windows 10 on an older PC and considering upgrading to Windows 11 despite the infamous compatibility checks?
Good news: it's absolutely possible, and this guide will walk you through it step-by-step – including how to bypass TPM and Secure Boot requirements using Rufus.
We'll start with some essential checks to ensure a smooth transition.



Step 0: Check Your System Compatibility

  1. Before diving into the installation, make sure your system meets the practical (not official) requirements for running Windows 11.
    • Press Win + R, type msinfo32 , and hit Enter.
  1. Look for BIOS Mode:
    • If it says UEFI – perfect!
    • If it says Legacy, you'll need to convert to UEFI before proceeding.

Retail License Check

  1. Open a terminal or powershell (cmd.exe) and type:
    • slmgr /dli and click OK
    • If it says Retail, you're good to go. This means your license is transferable and will re-activate after reinstalling.


Step 1: Download the Windows 11 ISO

    • Go to the official Microsoft Windows 11 download page.
    • Scroll down to Download Windows 11 Disk Image (ISO).
    • Select the latest edition, click Download.
    • Choose your language and download the 64-bit ISO.
